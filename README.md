# zmjjrr's Blog

这是我的个人技术博客，使用 Jekyll 和 GitHub Pages 搭建。

## 博客内容

本博客主要分享：

- CTF（Capture The Flag）比赛writeup
- 二进制漏洞利用技术
- PWN和逆向工程经验
- 安全研究心得

## 本地开发

如果你想在本地运行此博客：

```bash
# 安装依赖
bundle install

# 本地运行
bundle exec jekyll serve

# 访问 http://localhost:4000
```

## 技术栈

- **Jekyll**: 静态网站生成器
- **Minima**: Jekyll主题
- **GitHub Pages**: 托管平台
- **Markdown**: 内容编写

## 文章列表

所有博客文章都存放在 `_posts` 目录中，文件名格式为 `YYYY-MM-DD-title.md`。
